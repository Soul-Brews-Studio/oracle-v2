# Lesson Learned: Learn Then Combine

**Date**: 2026-01-17
**Context**: KlakMath + Gesture Control Demo Session
**Category**: Workflow Pattern

---

## The Pattern

```
/learn repo-A → /learn repo-B → combine A + B = something new
```

## What Happened

ใช้ `/learn` กับ 2 repos:
1. **KlakMath** - Unity math library (CdsTween, XXHash, Noise)
2. **mission-03-gesture-control** - MediaPipe hand tracking

จากนั้นรวมทั้งสองเข้าด้วยกัน → Interactive Three.js demo ที่ควบคุมด้วยมือ

## Why It Works

1. **Learn creates portable knowledge** - Documentation ใน ψ/learn/ พร้อมใช้งานเสมอ
2. **Concepts transfer across languages** - CdsTween จาก C# → JavaScript ได้
3. **Combining is creative** - การรวม 2 สิ่งที่ไม่เกี่ยวกันสร้าง value ใหม่

## The Formula

```
Learning A (math/algorithms)
+ Learning B (input/interaction)
= Interactive Experience
```

## Application

ครั้งหน้าเมื่อต้องสร้างอะไรใหม่:
1. หา 2-3 repos ที่มี concepts ที่ต้องการ
2. ใช้ /learn เก็บ knowledge
3. Combine concepts ในงานใหม่

## Key Insight

> `/learn` ไม่ใช่แค่ documentation - มันคือ building blocks สำหรับ future projects
